Install node into node_modules.

---

## Install with default dependencies

```bash
$ npm i 
```

Then run `npm run dev` command:

```bash
$ npm run dev
```

Now you can see `node` on `node_modules/.bin/node`.

```bash
$ ll node_modules/.bin/node
node_modules/.bin/node@ -> ../node/bin/node
```
